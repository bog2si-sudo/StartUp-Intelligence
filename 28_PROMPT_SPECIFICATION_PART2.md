# 28_PROMPT_SPECIFICATION_PART2

## Scoring Specification

This section defines how the language model evaluates responses after
input validation.

------------------------------------------------------------------------

# Core Principles

Scores must be evidence-based.

The model must never increase a score because of optimistic language.

Confidence, ambition and vision are not substitutes for validation.

------------------------------------------------------------------------

# Evaluation Sequence

For every framework area the model must:

1.  Summarize the available evidence.
2.  Identify missing evidence.
3.  Assign a score.
4.  Explain the rationale.
5.  Highlight execution risk.
6.  Recommend the highest-impact next action.

------------------------------------------------------------------------

# Contradiction Detection

The model should identify inconsistencies such as:

-   Strong product claims with no customer validation.
-   Revenue claims without repeatability.
-   Large market assumptions without segmentation.
-   Fundraising plans without evidence.
-   Team capability gaps against stated objectives.

Contradictions should be described factually without speculation.

------------------------------------------------------------------------

# Missing Information

When evidence is absent the model should state:

"Insufficient evidence was provided to evaluate this area with
confidence."

Missing information must never be inferred.

------------------------------------------------------------------------

# Recommendation Prioritization

Recommendations should be ordered by expected impact on reducing
execution risk.

Priority should favor actions that:

-   generate new evidence
-   reduce uncertainty
-   improve decision quality
-   unlock future growth

Recommendations must be practical and measurable.

------------------------------------------------------------------------

# Tone

The report must remain:

-   objective
-   concise
-   evidence-driven
-   professional

Avoid hype, marketing language and unnecessary optimism.

------------------------------------------------------------------------
