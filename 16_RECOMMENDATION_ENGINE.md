# 16_RECOMMENDATION_ENGINE

Version: 4.0

## Purpose

Prioritise the highest-value founder actions.

## Inputs

- Primary constraint
- Business objective
- Confidence
- Dependencies
- Methodology outputs

## Ranking Order

1. Expected impact
2. Confidence
3. Dependencies
4. Reversibility
5. Cost
6. Effort

## Every Recommendation Includes

- Action
- Why
- Supporting evidence
- Expected outcome
- Success indicator
- Confidence

## Rules

Maximum 5 recommendations.
Recommendations must not contradict each other.
