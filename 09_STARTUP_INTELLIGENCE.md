# 09_STARTUP_INTELLIGENCE

Version: 4.0

## Purpose

Generate the paid Founder Intelligence Brief.

## Processing Pipeline

1. Validate assessment
2. Resolve benchmarks
3. Execute methodologies
4. Identify primary constraint
5. Calculate confidence
6. Generate recommendations
7. Build prompt
8. Generate report
9. Create PDF
10. Email founder

## Deliverables

- Executive Investment Thesis
- Benchmark Context
- Transparent Valuation
- Constraint Analysis
- Investor Perspective
- Recommendations
- 30/90/180 Day Roadmap
- Methodology
- Confidence

## Success

The founder understands where the next unit of time or capital is most likely to create value.
