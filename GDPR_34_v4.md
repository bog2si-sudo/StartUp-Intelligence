# 34_GDPR

Version: 4.0

Status: Production

------------------------------------------------------------------------

# Purpose

This document defines the GDPR compliance requirements for the Founder
Intelligence Platform.

The platform processes founder-submitted business information and
personal data. All processing must comply with applicable GDPR
requirements.

------------------------------------------------------------------------

# Personal Data Processed

The platform may process:

-   Founder name
-   Email address
-   Startup name
-   Country
-   Uploaded documents
-   Form responses
-   Payment metadata
-   Technical request metadata

Only data necessary for delivering the service should be collected.

------------------------------------------------------------------------

# Processing Principles

Processing must be:

-   lawful;
-   transparent;
-   purpose-limited;
-   data-minimised;
-   accurate;
-   securely stored;
-   retained only as long as necessary.

------------------------------------------------------------------------

# Legal Basis

Processing is based on:

-   user submission;
-   performance of the requested service;
-   legitimate operational interests where applicable.

------------------------------------------------------------------------

# Storage

Data is stored using Supabase.

Reports and uploaded files must not be publicly accessible.

Private report access must use high-entropy tokens.

------------------------------------------------------------------------

# User Rights

Users must be able to request:

-   access;
-   correction;
-   deletion;
-   export of their personal data.

Requests should be processed without undue delay.

------------------------------------------------------------------------

# Security

Protect personal data using:

-   HTTPS
-   encrypted storage where applicable
-   restricted access
-   server-side secret management

------------------------------------------------------------------------

# Third-Party Processors

Current processors include:

-   Vercel
-   Supabase
-   Tally
-   Stripe
-   OpenAI
-   Resend

Users should be informed that these providers participate in service
delivery.

------------------------------------------------------------------------

# Retention

Retain data only for operational and legal requirements.

Delete or anonymise records when no longer required.

------------------------------------------------------------------------

End of document.
