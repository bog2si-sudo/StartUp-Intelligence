# 29_REPORT_TEMPLATE_PART1

Version: 4.0

Status: Production

------------------------------------------------------------------------

# Purpose

This document defines the first section of the production report
template.

The application assembles deterministic data and invokes the language
model to populate the narrative defined by this template.

------------------------------------------------------------------------

# Report Header

Display:

-   Founder Intelligence Platform
-   Report Type
-   Assessment ID
-   Startup Name
-   Assessment Date
-   Report Version

The application populates these values.

------------------------------------------------------------------------

# Executive Summary

## Objective

Provide a concise summary of the assessment.

The narrative must include:

-   overall readiness;
-   strongest evidence;
-   primary execution constraint;
-   highest-priority recommendation;
-   confidence statement.

Maximum length: 250 words.

------------------------------------------------------------------------

# Current Position

Describe the startup's current position using only supplied inputs.

Include:

-   current stage;
-   target customer;
-   product maturity;
-   commercial progress;
-   overall execution context.

Do not introduce assumptions.

------------------------------------------------------------------------

# Readiness Overview

The application provides deterministic scores.

Display:

-   Overall Readiness Score
-   Evidence Quality
-   Confidence Level

These values must be rendered exactly as supplied.

------------------------------------------------------------------------

# Dimension Summary

Provide narrative for each deterministic dimension:

-   Market Validation
-   Product Readiness
-   Customer Evidence
-   Commercial Repeatability
-   Team Readiness
-   Fundraising Readiness

Each explanation should reference only supplied evidence.

------------------------------------------------------------------------

End of Part 1.
