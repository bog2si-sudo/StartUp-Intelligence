# 29_REPORT_TEMPLATE_PART3

## Action Plan and Closing Assessment

This document defines the final section of the Founder Intelligence
Platform report template.

------------------------------------------------------------------------

# 30-Day Plan

Focus on the highest-priority validation activities.

For each action include:

-   Objective
-   Owner
-   Expected outcome
-   Success metric

------------------------------------------------------------------------

# 60-Day Plan

Build on completed validation work.

Focus areas may include:

-   Customer discovery
-   Product iteration
-   Pricing validation
-   Go-to-market refinement

Each action should remain measurable.

------------------------------------------------------------------------

# 90-Day Plan

Emphasize execution readiness and scalable growth.

Typical objectives include:

-   Repeatable customer acquisition
-   Improved unit economics
-   Stronger operating cadence
-   Investor readiness

------------------------------------------------------------------------

# Closing Assessment

Provide a concise concluding assessment that:

-   Summarizes the startup's current readiness.
-   Reinforces the most important evidence gaps.
-   Highlights the next critical milestone.
-   Encourages disciplined, evidence-based execution.

The conclusion must remain objective and should not guarantee future
outcomes.

------------------------------------------------------------------------

# Report Metadata

Append the following metadata:

-   Assessment ID
-   Report Version
-   Prompt Version
-   Scoring Model Version
-   Generated Timestamp

This metadata supports auditability and future report comparisons.

------------------------------------------------------------------------

End of Part 3.
