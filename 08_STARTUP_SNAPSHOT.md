# 08_STARTUP_SNAPSHOT

Version: 4.0

## Purpose

Generate a free assessment that answers one question:

'What should I do next?'

## Inputs

- Founder evidence
- Startup profile
- Objective
- Current constraint

## Outputs

- Executive Summary
- Current Position
- Primary Constraint
- Missing Evidence
- One Highest-Leverage Action
- Confidence

## Rules

- No valuation.
- Maximum one recommendation.
- Reading time under five minutes.
