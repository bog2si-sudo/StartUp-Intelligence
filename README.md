# Founder Intelligence Platform (V4)

Status: Production Specification

## Purpose

This repository contains the complete implementation specification for the Founder Intelligence Platform.

The platform helps founders make better business decisions by combining:

- Founder-provided business evidence
- Public benchmark evidence
- Transparent business methodologies
- Deterministic reasoning
- AI-generated narrative

## Technology Stack

- Frontend: Next.js
- Hosting: Vercel
- Forms: Tally
- Payments: Stripe (via Tally)
- Database: Supabase
- Storage: Supabase Storage
- Email: Resend
- AI: OpenAI
- Source Control: GitHub
- Development: Cursor

## Repository Principles

- Business rules are deterministic.
- The LLM never performs calculations.
- Reports are reproducible.
- All conclusions are evidence-based.
